## üöÄ Better Clipboard v1.0.0

This is an enhanced clipboard manager for Windows. It stores text and images from your clipboard history, allows multiple selection with Ctrl+Shift+arrows, and pastes them automatically on Enter.

## üìã Requirements
- Windows 10 or 11 (64-bit).
- No additional installations needed (includes .NET runtime).
- Run as administrator for hotkeys and keyboard simulation to work.

## üõ†Ô∏è How to Run
1. Extract the ZIP and run BetterClipboard.exe as administrator (right-click ‚Üí "Run as administrator").
2. The app starts in the system tray (notification area, check hidden icons).
3. Use Ctrl+Alt+C to open the clipboard window.

## üîë Usage
- Copy text/images (Ctrl+C) ‚Äî they appear in history.
- Open with Ctrl+Alt+C.
- Navigate with arrows, select multiple with Ctrl+Shift.
- Press Enter to paste into the active input field.
- Esc or click outside to close.

## üîÑ Autostart
1. Create a shortcut to BetterClipboard.exe.
2. In shortcut properties: Compatibility ‚Üí "Run this program as an administrator".
3. Place the shortcut in Startup folder (Win+R ‚Üí shell:startup).

## ‚ùì Troubleshooting
- If no tray icon: Check Task Manager for process, run as admin.
- Hotkey issues: Ensure admin mode.
- Log: Check log.txt in the app folder for errors.
